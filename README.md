🏨 Hospitality Performance Insights Dashboard
A Power BI dashboard providing a comprehensive view of hotel performance across multiple cities, room categories, and booking platforms. This project helps hospitality businesses monitor revenue, occupancy, customer behaviour, and operational efficiency through interactive visual analytics.

📌 Project Overview
This dashboard consolidates key performance metrics from multiple hotel properties to support data‑driven decision‑making. It highlights trends, compares performance across properties, and identifies opportunities to improve revenue and occupancy.

📊 Key Features
1. KPI Summary
Revenue: Total revenue generated across properties

RevPAR: Revenue per available room

Occupancy %: Percentage of rooms occupied

ADR: Average daily rate

Realisation %: Revenue realisation after cancellations

DSRN: Daily sellable room nights

Download the .pbix file from this repository.

Open it in Power BI Desktop.

Use the slicers (City, Room Type, Month, Week) to explore insights.

Modify or extend the dashboard as needed.
